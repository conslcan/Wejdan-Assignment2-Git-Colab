# Titanic Survival Classification

This project builds and evaluates classification models to predict Titanic passenger survival.

## Contents
- `titanic_survival_classification.ipynb` — main analysis notebook
- `data/titanic.csv` — Titanic dataset
- `figures/` — saved plots and visualizations generated during the analysis

## Models
Logistic Regression and Random Forest classifiers were trained and compared, 
achieving 76-82% accuracy and 0.83-0.85 ROC-AUC. Sex, passenger class, and fare 
were the strongest predictors of survival across both models.
